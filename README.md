# Thread Pool Server – Sci-Fi GUI

## Overview
This project is a multithreaded Thread Pool Server developed with a Sci-Fi styled graphical interface for visualizing concurrent task execution in real time.

The application demonstrates how thread pools manage and execute multiple tasks efficiently using worker threads while providing live execution logs, timeline visualization, task monitoring, and CPU benchmarking.

The GUI is designed using pure Win32/GDI concepts with a futuristic terminal-inspired interface.
